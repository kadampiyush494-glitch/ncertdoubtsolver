# NCERT RAG Backend

This project is a **FastAPI backend** implementing a **Retrieval-Augmented (RAG) pipeline** over **NCERT PDFs**.  
The backend ingests NCERT textbooks, generates embeddings, stores them in FAISS, and exposes APIs to retrieve relevant textbook content based on user queries.
